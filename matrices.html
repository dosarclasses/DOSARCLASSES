01 · Foundations

What is a Matrix?

A matrix is simply a rectangular arrangement of numbers, organized into rows and columns and enclosed in brackets. Matrices let us handle whole tables of related numbers — like sales figures across cities and products — as a single mathematical object, with their own rules for addition, subtraction, and multiplication.

Why does it matter?

Once numbers are arranged into a matrix, entire systems of data can be combined in one operation instead of many. But matrices don't behave exactly like ordinary numbers — you can't always add or multiply any two of them, and even when you can multiply them, the order you multiply in usually matters. Learning these rules is the heart of this chapter.

Where it's used

  • Organizing tabular data (sales, marks, inventory)
  • Representing systems of linear equations
  • Computer graphics and image transformations
  • Encoding and solving real-world "combination" problems
  • Foundations for higher studies in Class 11–12

12 · One Page

Quick Revision

Key Facts

Order = rows × columns

Addition/Subtraction: same order only

Multiplication: A(m×n)×B(n×p) → order m×p

Transpose: rows and columns interchanged

Rules to Remember

AB ≠ BA in general (not commutative)

AI = IA = A (identity matrix property)

Equal matrices: same order AND same elements

All Definitions & Rules (memorize this)

Exam Tips

  • Before adding or subtracting, always check that both matrices have exactly the same order.
  • Before multiplying, always check that the columns of the first matrix equal the rows of the second.
  • When solving for unknowns, write both sides in the same order and compare corresponding elements one by one.
  • Never assume AB = BA — always compute both separately unless the question tells you they are equal.

ICSE Class 10 Mathematics

Order & Types

Order = (rows) × (columns)

Row matrix: 1×n · Column matrix: m×1

Square matrix: rows = columns

Zero matrix: all elements 0

Identity matrix: 1's on diagonal, 0's elsewhere

Operations

Addition/Subtraction: same order, combine corresponding elements

Scalar multiplication: multiply every element by k

Matrix multiplication: A(m×n)×B(n×p) → order m×p

Key Properties

AB ≠ BA in general

AI = IA = A

(A^T)^T = A

(A+B)^T = A^T + B^T

Equal matrices: same order AND same elements
